This project was created as part of the course: https://www.codecademy.com/courses/learn-intermediate-java/.

It is a restaurant simulator, showing understanding of issues, among others, in the field of classes, objects, pipelines or interface implementation.

After compiling and running the program, you need to enter the name and money on the basis of which the Customer class object is created. Next, an object of the TakeOutSimulator class is initialized, enabling the selection of options from the menu, completed with the checkoutCustomer method.
